Super-Gaussian Transverse Profile
=================================

Used to define a super-Gaussian transverse laser profile.

The shape of the profile is characterised by the waist :math:`w_0` and by one "order parameter" :math:`n`, where :math:`n=2` gives a standard Gaussian profile, and the profile converges to a flat-top when :math:`n` goes to infinity.
